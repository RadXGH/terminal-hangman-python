# terminal-hangman-python
Simple and basic hangman game played in terminal (doesn't have any fancy GUIs)

## Features
- Starting screen
- Loading screen
- Detects keyboard key presses

## Dependencies
- Made in Python 3.9.5
- Random module
- Time module
- msvcrt (MS VC++ runtime) module
- OS module
- Collections module
